1. Создал интерфейс ChatMediator для отправки сообщений и добавления пользователей.
2. Реализовал класс ChatRoom как медиатор для управления пользователями и передачей сообщений.
3. Определил абстрактный класс User с методами для общения.
4. Реализовал классы RegularUser и PremiumUser для отправки и получения сообщений.
5. В классе Main показал работу чата с пользователями и их общением.

Mediator pattern позволяет объектам общаться через центральный медиатор вместо прямого взаимодействия.
Это упрощает систему и снижает связанность между объектами.
