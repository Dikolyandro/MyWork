1. Создал абстрактный класс ReportGenerator с шаблонным методом generateReport.
2. Определил абстрактные методы для форматов заголовка, тела и подвала.
3. Реализовал классы PDFReportGenerator и HTMLReportGenerator для специфичных форматов.
4. Создал класс ReportingSystem для генерации отчетов.
5. В классе Main продемонстрировал создание различных отчетов.

Template Method pattern определяет общий процесс в базовом классе, а детали реализует
в подклассах. Это сохраняет последовательность, позволяя менять только нужные части.
