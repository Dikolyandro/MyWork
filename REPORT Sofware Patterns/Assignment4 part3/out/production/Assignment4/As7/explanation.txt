1. Создал класс Document для редактирования и сохранения версий.
2. Реализовал класс DocumentVersion для хранения состояния документа.
3. Создал класс VersionControl для управления версиями документа.
4. В классе Main показал работу системы с сохранением и восстановлением версий.

Memento pattern позволяет сохранять и восстанавливать состояние объекта.
Это удобно для создания версий и возврата к предыдущим состояниям.
