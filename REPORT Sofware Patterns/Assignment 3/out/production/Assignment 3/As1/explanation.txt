1.В этом задании я использовал паттерн проектирования "Adapter", чтобы добавить поддержку новых
форматов аудио в наш музыкальный плеер, который изначально воспроизводил только MP3-файлы.
2.Я создал интерфейс для плеера и реализовал класс для проигрывания MP3. Затем я добавил интерфейсы
для WAV и AAC форматов и реализовал их в отдельном классе.
3.Чтобы объединить всё это, я создал адаптер, который позволяет основному плееру работать с новыми форматами.
4.В итоге плеер теперь может воспроизводить MP3, WAV и AAC файлы, не изменяя его исходную структуру.

Адаптер — это шаблон, который помогает соединить несовместимые части программы, чтобы
они могли работать вместе. Он как переходник, который переводит один тип интерфейса в
другой, не меняя существующий код.
