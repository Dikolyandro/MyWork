1. Я использовал паттерн Facade, чтобы создать систему управления умным домом.
2. Создал классы для устройств, таких как свет, термостат, система безопасности
и развлекательная система.
3. Реализовал класс `SmartHomeFacade`, который объединяет управление всеми
устройствами и предоставляет простые методы.
4. В классе `SmartHomeApp` показал, как легко управлять системой с помощью одного интерфейса.
5. Это позволяет пользователям контролировать все устройства без необходимости
взаимодействовать с каждым из них по отдельности.

Паттерн Facade предоставляет простой интерфейс для работы с комплексной системой,
скрывая сложность её внутренней структуры. Это позволяет пользователям легко взаимодействовать с
системой, не углубляясь в детали реализации каждого компонента.
