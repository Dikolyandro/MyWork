1. В этом задании я использовал паттерн "Bridge", чтобы создать универсальный пульт
для управления разными устройствами, такими как телевизор, DVD-плеер и звуковая система.
2. Сначала я сделал интерфейс для устройств с основными методами, такими как включение/выключение,
изменение канала и регулировка громкости.
3. Затем я создал классы для конкретных устройств, которые реализуют этот интерфейс.
4. После этого я реализовал абстрактный класс пульта, в который можно "подключить" любое
устройство через интерфейс.
5. Также я добавил конкретные реализации пульта — базовый и продвинутый, с дополнительными
функциями, например, выключением звука.
6. В итоге один пульт можно использовать с разными устройствами без изменения кода устройств,
что делает систему гибкой и удобной.

Паттерн Bridge разделяет логику управления и сами устройства, чтобы их можно было менять
независимо друг от друга. Это делает код гибче, потому что новые устройства или новые
функции управления можно добавлять без изменений в уже существующем коде.
